# BusTrace-Project
Modern animated developer portfolio built with React, Vite, GSAP, and Framer Motion featuring responsive design, smooth scroll interactions, and performance-focused UI engineering.

## Run Locally

Open terminal and run:

```bash
cd C:\Users\kafka\OneDrive\Desktop\BusProject
cd bus-tracker
node server.js
```

## Tech Stack

* Node.js
* JavaScript

## Features

* Real-time bus tracking
* Route monitoring
* Lightweight local server setup

## Start the Project

After running the commands above, the local server will start and the project will be available in your browser.
